Dark Mode Toggle

A complete implementation using Tailwind CSS and vanilla JavaScript with localStorage persistence and system preference detection.

System Preference

Automatically detects your system's dark mode preference on first visit.

Persistent Storage

Your theme preference is saved and restored on page refresh.

Smooth Transitions

Elegant animations make theme switching feel seamless and polished.

Key Features

  • Tailwind CSS CDN integration with no build process required
  • Vanilla JavaScript implementation with localStorage persistence
  • System preference detection using prefers-color-scheme media query
  • Accessibility features including ARIA labels and keyboard navigation
  • Flash prevention by applying theme before page renders
